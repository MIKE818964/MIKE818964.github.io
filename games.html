🎮 GAMES & OVERLAYS

The stuff we build when the work's done.

Companions, overlays, and mods for the games we actually play — an EVE Online command deck, a Diablo IV overlay, a WoW UI suite, an arcade frontend, and an original card game. The same polish bar as everything else, pointed at fun.

🛡️ Ban-safe by design — the whole point. Every overlay here is read-only and client-side: it reads public data or your own account's data, renders in its own window, and never injects input, never broadcasts one keypress to many clients, never reads game memory. One human input drives one client. That line is the personality, not a disclaimer.
THE LINEUP

Built, verified, and legal.

FREE · SHIPPED · OPEN SOURCE

EVE Fleet Foreman

Ranks every ore by ISK/m³ off live Jita prices, computes fleet yield and time-to-fill, and splits mined loot fairly. Pro tier pulls your real 30-day Mining Ledger via EVE's own SSO login. Already public on GitHub with a Windows installer.

Ban-safe: reads public market data + (with your login) your own ESI ledger. Never touches the game client. CCP-sanctioned ESI pattern.
BUILT · LIVE-TESTED

BADGER — the EVE command deck

The overlay that makes EVE more fun: a best-in-class window switcher with live thumbnails, plus threat radar, pilot-intel cards, "what can I build / fly," net-worth HUD, kill-streak juice, and an in-app voice "Ship AI" you talk to. A game on top of your game.

Ban-safe: window management + overlay only. Switches focus between clients, never sends keys into them. No input broadcasting — ever. Voice is local mic/speaker.
🩸
BUILT · VERIFIED LIVE

Sanctuary Watch · Diablo IV

A blood-gothic transparent overlay for Diablo IV: live World Boss / Helltide / Legion / Whisper timers, an editable build guide, BiS-loot and aspect checklists, paragon quick-ref, and a session/death tracker — all in a WebGL blood-drip UI.

Ban-safe: 100% read-only. A separate always-on-top window rendering our own timers and checklists — never touches the D4 process, memory, or files.
⚔️
LIVE IN-GAME

CrimsonUI · World of Warcraft

A from-scratch dark-gothic WoW addon suite: themed unit frames, Diablo-style floating damage numbers, a DPS meter, a multi-kill announcer, a mining map-pin overlay — and a macro sequencer that beat GSE and GRIP on this machine when both broke.

Ban-safe: client-side addon using only Blizzard's permitted API. The sequencer is the legal one-press model — you still press the key each cast; no automation, no injection.
🕹️
SELLABLE · COMMERCIAL

Retro Arcade Framework

A Windows-packaged arcade-cabinet frontend — a polished fullscreen tile-grid launcher over 15 systems via RetroArch cores, controller support, a ROM scanner, and a real installer. Sell the frontend; the buyer brings their own games.

Legal by design: framework only — we ship no copyrighted ROMs. Bundled content is an attributed legal homebrew pack. It wraps RetroArch, it doesn't reimplement games.

Clean to publish: our own IP, standalone game, no anti-cheat surface. If sold: published pack odds, direct-buy option, no cash-out — monetization done honestly.
