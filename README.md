# mike818964.github.io

This repo hosts the [HONEYBADGER SOFTWARE INC landing page](https://mike818964.github.io)
on GitHub Pages.

It's a pure static site — no Jekyll, no build step. Just `index.html`,
`assets/css/style.css`, and a couple of remote-hosted images (the avatar
from GitHub's CDN).

To preview locally:

```
python -m http.server 8000
```

Then open http://localhost:8000.

To deploy: push to `main`. GitHub Pages auto-rebuilds within ~1 minute.
